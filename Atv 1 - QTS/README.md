# Projeto de Testes com Pytest

## 📌 Descrição
Este projeto tem como objetivo testar um sistema de operações matemáticas utilizando testes automatizados com a biblioteca pytest.

Foram aplicados conceitos de:
- testes unitários
- uso de fixtures
- validação de diferentes tipos de entrada
- identificação de defeitos
- análise de qualidade com base na norma ISO/IEC 25010
